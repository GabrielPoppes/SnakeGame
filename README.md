## About this Project
A game to exercise the logic of reasoning
  
### Installing

**Cloning the Repository**

```
$ git clone https://github.com/GabrielPoppes/SnakeGame

$ cd SnakeGame
```

**Installing dependencies**

```
$ yarn
```

_or_

```
$ npm install
```
